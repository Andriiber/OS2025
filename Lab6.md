| Назва команди | Її призначення та функціональність |
| ----- | ----- |
| mkdir mybackups  | Створення нової директорії **mybackups** у домашньому каталозі користувача |
| tar \-cvf mybackups/udev.tar /etc/udev | Команда **tar** використовується для об’єднання кількох файлів в один файл. В даному випадку вміст директорії  **/etc/udev**  буде збережено в архів **udev.tar** у директорії **mybackups**. Параметр **\-c** повідомляє команді tar створити файл tar. Параметр **\-v** означає "verbose", який наказує команді tar продемонструвати, що вона робить. Параметр **\-f** використовується для вказівки назви файлу tar. |
| ls mybackups | Виводить список файлів і тек у директорії mybackups, щоб перевірити, що архів udev.tar створено. |
| tar –tvf mybackups/udev.tar | Виводить список (вміст) файлів, що знаходяться всередині архіву udev.tar, без розпаковування. |
| tar –zcvf mybackups/udev.tar.gz /etc/udev | Створює стиснений архів (tar.gz) з каталогу /etc/udev і зберігає його у mybackups. |
| tar –xvf udev.tar.gz | Розпаковує вміст архіву udev.tar.gz у поточну директорію. |
| ls etc/udev/rules.d| Показує файли правил udev, які зберігаються в директорії rules.d |
| tar -rvf udev.tar /etc/hosts | Додає файл /etc/hosts до вже існуючого архіву udev.tar. |
| tar –tvf udev.tar | Виводить список усіх файлів, що тепер містяться в архіві udev.tar, включно з новододаним /etc/hosts. |
| ls -l words.gz | Показує детальну інформацію про файл words.gz (права доступу, власник, розмір, дата створення). |
| gunzip words.gz | Розпаковує файл words.gz, видаляючи оригінальний архів після успішного розпакування. |
| ls -l words | Показує детальну інформацію вже про розпакований файл words — його розмір, дату, права тощо. |
| bzip2 words | Стискає файл words за допомогою алгоритму bzip2. |
| ls -l words.bz2 | Показує детальну інформацію про новостворений стиснений файл words.bz2, у тому числі його менший розмір. |
| bunzip2 words.bz2 | Розпаковує файл words.bz2, відновлюючи початковий файл words. |
| xz words | Стискає файл words за допомогою алгоритму XZ. |
| ls -l words.xz | Показує детальну інформацію про створений стиснений файл words.xz — розмір, дату створення, права доступу тощо. |
| unxz words.xz | Розпаковує файл words.xz, відновлюючи оригінальний файл words. |
| zip words.zip words | Створює стиснений архів words.zip, який містить файл words. |
| ls -l words.zip | Виводить детальну інформацію про архів words.zip — його розмір, дату створення, права доступу тощо. |
| zip -r udev.zip /etc/udev | Створює стиснений архів udev.zip, включаючи всі файли та підкаталоги з каталогу /etc/udev. |
| ls -l udev.zip | Показує детальну інформацію про архів udev.zip: розмір, права доступу, дату створення. |
| unzip -l udev.zip | Показує список усіх файлів і папок всередині архіву udev.zip, не розпаковуючи його. |
| rm -r etc | Видаляє директорію etc разом з усіма підкаталогами та файлами всередині неї. |
| unzip udev.zip | Розпаковує архів udev.zip у поточну директорію, відновлюючи папку etc та всі файли з /etc/udev, які були в архіві. |
