- Вводити, редагувати, видаляти текст (редактори файлів).

У Linux існує багато текстових редакторів, які працюють у терміналі. Найпоширеніші:

Приклади програм

    •	nano — простий, базовий редактор.
    
    •	vim — потужний редактор із режимами роботи.
    
    •	emacs — розширюваний редактор із величезним функціоналом.
    
    •	micro — сучасний редактор із зручною навігацією.
    
    •	joe — легкий та простий редактор.



- Здійснювати моніторинг процесів та ресурсів системи (аналог диспетчера задач або системного монітора в графічній оболонці).

    •	htop — покращена версія top з кольоровим інтерфейсом.

<h1>Те що знайшов я:</h1>

1. Команда oneko

Команда oneko додає на екран термінала кошеня, яке слідуватиме за вашим курсором.

Спочатку потрібно встановити oneko:

$ sudo apt install oneko

А потім можна використовувати:

$ oneko

2. Команда aafire

Ви коли-небудь замислювалися, який вигляд має вогонь на чорному фоні термінала? Завдяки команді aafire ви зможете це побачити.

Спочатку потрібно встановити aafire:

$ sudo apt install libaa-bin

А потім можна використовувати:

$ aafire
